# CRUD API MySQL - Node - Express  

Simple api using node - mysql. 
**Sorry for the interface :/, I focused on the backend**

## Instalation

Install all dependencies

```
npm install
```

Database on file/**api-mysql.sql**

Run app

```
npm start
```
